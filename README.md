## Featured Projects

### OrientAI — Multimodal Cognitive Support Assistant

A multimodal AI-powered assistant designed to support individuals with mild to moderate dementia.

My contribution focuses on **Scrum coordination, synthetic patient personas, prompt design, RAG test scenarios, hallucination test cases and demo data preparation**.

**Focus:** Multimodal AI, RAG, Prompt Engineering, Synthetic Data, Healthcare-Oriented AI Prototype

---

### Vision Assistant — AI Assistant for Visually Impaired Users

A real-time visual assistant that provides scene understanding, object detection, OCR, visual question answering and voice-based interaction.

**Focus:** Computer Vision, VLMs, OCR, YOLO, TTS/STT, FastAPI

---

### DocSage — Document Understanding and QA System

A document-based question answering system that allows users to upload documents and ask questions grounded in document content.

**Focus:** NLP, RAG, Embeddings, Document QA, FastAPI, Streamlit

---

### FakeSense — Fake News Classification

A machine learning project for fake news detection using deep learning-based text classification.

**Focus:** NLP, LSTM, Text Classification, Model Evaluation

---

### Speaker Diarization System

A speaker diarization project comparing diarization pipelines and visualizing speaker timelines, durations and evaluation metrics.

**Focus:** Speech AI, Speaker Diarization, Pyannote, Resemblyzer, SpeechBrain
